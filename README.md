## How to run
- Install simplehttpserver
https://www.npmjs.com/package/simplehttpserver
```
npm install simplehttpserver -g
```

- Run the server locally
```
simplehttpserver ./
```

## How to run it without termination in openstack
nohup simplehttpserver ../ &
